## Getting Started

We use npm for the package manager so run
```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
We are using eslint and prettier for code style infrocment so you can use

```bash
npm run lint
```
to automatically run files

## Learn More
